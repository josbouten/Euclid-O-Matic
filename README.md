Euclid-O-Matic is a 4 channel 16 step euclidean sequencer for eurorack.
There have been many changes since the first version was built.
v0.1 started out from a publication by L J Brackney of Suncoast Polytechnical High School called
"Euclidean Rhythm Generator Kosmo/Eurorack" dated 11/21/2021 which I implemented
in eurorack format. However starting in 2022 the software was almost completely
rewritten leading up to v0.2 which is much more feature rich. 
V0.1 is not supported anymore as of june 2026.
